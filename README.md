# Retail Sales Performance Analysis

## Project Overview

This project analyzes retail sales data to identify revenue trends, product performance, return patterns, and business growth opportunities using Python and data visualization techniques.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Dataset

The project uses retail sales datasets containing product-wise sales information and monthly sales records.

## Key Findings

* Basket category generated the highest revenue.
* Art & Sculpture was the second highest-performing category.
* Basket and Art & Sculpture showed the highest return values.
* Several product categories recorded zero returns.
* Sales showed growth trends over time, with significant peaks during certain periods.
* Revenue was concentrated in a few high-performing product categories.

## Visualizations

* Top Products by Net Sales
* Returns Analysis
* Monthly Sales Trends
* Correlation Heatmap

## Conclusion

The analysis highlights high-performing product categories, return behavior, and sales growth patterns that can help businesses improve decision-making and optimize revenue generation.
